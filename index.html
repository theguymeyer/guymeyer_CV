SerpUI

A web-based search engine for people with visual impairment. SerpUI pulls real-time results from the Google Custom Search Engine API and presents them with a new interface better suited for people with visual impairment. To add functionality you can connect a Nintendo JoyCon controller for interactive browsing. Our product has also been tested with young users with the hopes of making Google more accessible to everyone! The new format separates the two types of data; search results and their content. The user can discover new results by moving up and down, and learn more about each result by moving left and right. Enjoy SERPing!

This interface was a lot of fun to make! Trying to put yourself in the shoes of others with such different perpectives is challenging to begin with. But when you're asked to find the best way to help them surf the web, things get interesting :)

Try it out for yourself: serpui.cas.mcmaster.ca (optimized for the Google Chrome Web Browser)
